## Installation

To install the project, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/xyzeez/steps.git
```

2. Navigate to the project directory:

```bash
cd steps
```

3. Install dependencies:

```bash
npm install
```

4. Start the application:

```bash
npm start
```
